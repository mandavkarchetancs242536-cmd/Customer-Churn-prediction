# Customer Churn Prediction System

## 📌 Project Description
This project predicts customer churn using Machine Learning. It uses Logistic Regression to determine whether a customer will leave or stay based on tenure and monthly charges.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib & Seaborn
- Tkinter (GUI)

## 📊 Features
- Data preprocessing and visualization
- Machine learning model for prediction
- User-friendly GUI for input and output
